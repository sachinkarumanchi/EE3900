\section*{Problem}
(3.8) The system function of a casual Linear time Invariant System is 
\begin{align}
    H(z)&=\frac{1-z^{-1}}{1+\frac{3}{4}z^{-1}}
\end{align}
The input of the system is 
\begin{align}
    x[n]&=\brak{\frac{1}{3}}^nu[n]+u[-n-1]
\end{align}
(a) Find impulse response of the system,$h[n]$\\
(c) Is the system stable?,That is $h[n]$, is absolutely summable?

\section*{Solution}
(a) To find the impulse response we need to find the inverse $\mathcal{Z}$ transform of $H(z)$
\begin{align}
    H(z)&=\frac{1-z^{-1}}{1+\frac{3}{4}z^{-1}}\\
    &=\frac{1}{1+\frac{3}{4}z^{-1}}-\frac{z^{-1}}{1+\frac{3}{4}z^{-1}}
\end{align}
Now, apply Inverse $\mathcal{Z}$ transform on both sides \\
We know the Inverse $\mathcal{Z}$ transform of $\frac{1}{1-az^{-1}}$ is $a^nu[n]$\\
By using Time shifting property and Inverse $\mathcal{Z}$ transform we obtain the impulse response
\begin{align}
    h[n]&=\brak{\frac{-3}{4}}^nu[n]-\brak{\frac{-3}{4}}^{n-1}u[n-1]
\end{align}
(b) For the system to be stable $h[n]$ must be absolutely summable \\
$h[n]$ is absolutely summable in ROC of $H(z)$ 
Therefore, $H(z)$ is stable for \abs{z}$>$3/4
